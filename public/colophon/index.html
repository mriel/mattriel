Typography

This site uses Inclusive Sans by Olivia King. I chose it because of its easy readability on both light and dark screens and the thought and accessibility that Olivia King has put into it. Plus it's just a really beautiful font, I really like it.

Technology

I've built this site using 11ty. Its super easy to use static site generator that lets you pretty much choose whatever templating language you like. I wanted to use only the bare minimum for my site for a couple of reasons. Speed and Accessibility. So I made the concious decision to use only HTML, CSS, Markdown, Nunjuck, and 11ty to put everything together.

Open Source

I'll constantly try and improve this site. You can always see how its going and how everything is built on my github.com/mriel/mattriel.
